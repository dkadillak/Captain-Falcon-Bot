# Captain-Falcon-Bot
Creating my own Captain Falcon AI in Super Smash Brothers Melee using the melee module

## What the bot does currently
It loops these actions:
* shorthop in place
* perform a forward ariel attack (called a F-air or Fair)
* fast falls the Fair
* L-Cancels
